## 🎯 使用流程

### 培训前（1周）

```
1. 阅读《员工培训手册》
   → 了解系统功能
   → 熟悉操作流程

2. 观看视频教程（如有）
   → YouTube链接或内网

3. 准备问题清单
   → 培训时提问
```

### 培训中（1小时）

```
1. 培训师按《培训演示大纲》讲解（30分钟）
   → 系统概述
   → 核心功能
   → 实战案例

2. 学员实操练习（30分钟）
   → 登录系统
   → 上传文档
   → 模拟问答

3. 考核评估（15分钟）
   → 填写《培训考核表》
   → 记录问题
```

### 培训后（1周）

```
1. 查阅《快速参考卡》
   → 日常操作参考
   → 解决常见问题

2. 实战使用
   → 至少使用3次
   → 记录体验

3. 反馈改进
   → 提交问题
   → 参加分享会
```

## 💡 培训技巧（给培训师）

### 讲解技巧

1. **用故事而非功能**
   - ❌ "这是语义搜索功能"
   - ✅ "张三用这个30秒找到了客户要的参数"

2. **演示胜过讲解**
   - 尽量现场操作
   - 让学员看到真实效果

3. **鼓励提问**
   - 没有愚蠢的问题
   - 问题是改进的机会

4. **关注慢的学员**
   - 不要让任何人掉队
   - 实操时巡视指导

### 互动技巧

1. **分组练习**
   - 2-3人一组
   - 互相帮助

2. **角色扮演**
   - 一人演客户
   - 一人用系统

3. **即时反馈**
   - 做对了立即表扬
   - 做错了耐心指导

4. **经验分享**
   - 邀请老用户分享
   - 真实案例更有说服力
