## 🌟 系统概述

### 什么是SRI动态销售情报系统？

**SRI（Sales Real-time Intelligence）** 是一套AI驱动的销售支持系统，帮助销售人员：

- 🤖 **AI客户问答** - 自动回答客户技术问题
- 🥽 **战术护目镜** - 实时分析客户意图，提供销售建议
- 📚 **技术武器库** - 存储产品手册、案例、参数等资料
- 🆘 **炮火支援** - 一键联系后方技术专家
- 📊 **数据看板** - 实时追踪项目和团队表现

### 系统架构

```

│           🎯 SRI 动态销售情报系统                   │
├─────────────────────────────────────────────────────┤
```

## 💡 核心功能详解

### 1. 📚 核心技术武器库

**用途：** 存储和管理所有技术文档

#### 如何上传文档

**步骤：**
1. 切换到"📚 核心技术武器库"标签
2. 选择"角色"（需要管理员权限）
3. 选择"文档分类"：
   - 📖 产品参数/白皮书
   - 📋 历史成功竞标书
   - 🎬 现场视频/照片
   - 🎙️ 专家答疑录音
   - 🏆 获奖证书/荣誉
   - 📊 案例集/分析报告

4. 上传文件（支持PDF、TXT、MP3、MP4）
5. 点击"🚀 一键向量化并装填入库"

**⚠️ 注意：**
- 文件大小建议 <10MB
- 音频/视频会自动转录为文本
- 上传后立即可用于AI问答

#### 如何搜索文档

**两种搜索方式：**

**🧠 语义搜索**（推荐）
- 输入："防腐涂料在海洋环境的应用"
- AI会理解你的意图，找到相关文档
- 支持同义词、近义词

**🔤 关键词搜索**（精确）
- 输入："WH-2026-A"
- 精确匹配文档中的关键词

**示例：**
```
查询："这个涂料能用多久？"
结果：找到 产品参数.pdf - "使用寿命15-20年"
```

### 3. 🥽 战术护目镜（销售密器）

**功能：** 实时AI战术分析（仅销售可见）

#### 它能做什么？

**实时分析客户意图：**
- 客户真正担心的是什么？
- 背后的动机是什么？
- 可能的顾虑和障碍？

**提供针对性建议：**
- 应该引用哪个案例？
- 可以承诺什么筹码？
- 如何化解客户顾虑？

#### 实际案例

**场景1：价格异议**

```
客户："你们的价格比同行贵太多了"

战术护目镜分析：
🎯 话术拆解：
客户担心性价比，认为价格高可能
影响整体成本和回报。

💡 出牌建议：
1. 强调产品独特价值和长期收益
2. 提供成功案例或用户反馈
3. 展示高品质带来的成本节约
4. 提供灵活付款方式或优惠方案
```

**场景2：技术疑虑**

```
客户："在海洋环境下真的能用15年吗？"

战术护目镜分析：
🎯 话术拆解：
客户对产品耐久性持怀疑态度，
需要可靠证据。

💡 出牌建议：
1. 引用【镇海炼化】8年运行数据
2. 展示耐盐雾测试报告（3000小时）
3. 提供延长保修承诺
4. 安排现场实地考察
```

## ❓ 常见问题

### Q1：AI回答不准确怎么办？

**A：** 有三种解决方案：

1. **上传更多文档**
   - AI基于武器库回答
   - 文档越多，回答越准确

2. **使用呼叫支援**
   - 点击"呼叫后方技术群"
   - 人工专家介入

3. **反馈给管理员**
   - 报告问题
   - 优化系统配置

### Q2：战术护目镜的建议可信吗？

**A：** 可信，但需要结合实际：

- ✅ AI分析基于真实文档
- ✅ 建议符合销售常识
- ⚠️ 最终决策权在销售人员
- 💡 建议：参考建议，结合经验

### Q3：客户会看到战术护目镜吗？

**A：** 不会！

- 🔒 战术护目镜仅销售可见
- 👁️ 客户只能看到左侧大屏
- 🛡️ 系统设计保护销售隐私

### Q4：如果网络断了怎么办？

**A：** 系统有离线预案：

- 📚 武器库数据本地缓存
- 📝 对话记录自动保存
- 🔄 网络恢复后自动同步

### Q5：可以在手机上使用吗？

**A：** 可以，但建议电脑：

- ✅ 手机浏览器可以访问
- ⚠️ 界面针对电脑优化
- 💡 客户演示建议用平板/笔记本
