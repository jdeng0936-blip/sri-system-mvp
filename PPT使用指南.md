## 📝 方式2：转换为PowerPoint（需要工具）

### 使用Marp转换

**安装Marp：**
```bash
# macOS
brew install marp-cli

# 或使用npm
npm install -g @marp-team/marp-cli
```

**转换为PPTX：**
```bash
cd ~/Desktop/销售AI情报系统/
marp 销售AI情报系统培训.md --pptx
```

**转换为PDF：**
```bash
marp 销售AI情报系统培训.md --pdf
```

## 📞 技术支持

### 遇到问题？

**文件打不开：**
- HTML文件：尝试不同浏览器（Chrome/Firefox/Safari）
- MD文件：检查Marp是否安装正确

**转换失败：**
- 确认Marp版本：`marp --version`
- 重新安装：`npm install -g @marp-team/marp-cli`

**样式不正常：**
- HTML版本：清除浏览器缓存
- MD版本：检查Markdown语法
