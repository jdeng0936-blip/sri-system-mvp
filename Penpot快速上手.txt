🎯 Penpot是什么？

  免费开源的移动应用原型设计工具
  ✅ 完全免费，无限项目
  ✅ 基于浏览器，无需安装
  ✅ 类似Figma，但开源
  ✅ 支持团队协作
  ✅ 手机扫码预览

🚀 5分钟创建第一个原型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：访问网站
  → 打开浏览器
  → 访问：https://design.penpot.app/

第2步：注册账号
  → 点击 "Sign up"
  → 输入邮箱和密码
  → 或用Google账号登录（更快）

第3步：创建项目
  → 点击 "Create new project"
  → 项目名：销售AI系统原型
  → 点击创建

第4步：创建手机画板
  → 按键盘 A 键
  → 选择 "iPhone 14 Pro" (393×852)
  → 或选择 "Android" (360×800)

第5步：开始设计
  → 按 R 键画矩形（顶部导航栏）
  → 按 T 键添加文字
  → 按 O 键画圆形（按钮）
  → 拖拽调整位置

第6步：手机预览
  → 点击右上角 "▶ Play" 按钮
  → 点击 "Share" 获取链接
  → 手机浏览器打开链接
  → 测试你的设计！

1. 顶部导航栏
   → 按 R 键创建矩形
   → 宽度：393px，高度：88px
   → 填充颜色：#1E40AF (蓝色)
   → 按 T 键添加文字："第一现场"
   → 文字颜色：白色

2. 输入框
   → 按 R 键创建矩形
   → 宽度：345px，高度：48px
   → 圆角：8px
   → 边框：1px 灰色
   → 添加占位文字："输入客户问题..."

3. 发送按钮
   → 按 R 键创建矩形
   → 宽度：120px，高度：48px
   → 圆角：8px
   → 填充颜色：#1E40AF
   → 添加文字："发送"

4. 结果卡片
   → 按 R 键创建矩形
   → 宽度：345px，高度：200px
   → 圆角：12px
   → 背景：白色
   → 阴影：0 2px 8px rgba(0,0,0,0.1)

🔗 添加交互（让原型"活"起来）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 进入原型模式
   → 点击右上角 "Prototype" 标签

2. 创建点击交互
   → 选择"发送"按钮
   → 拖拽蓝色圆点到下一个页面
   → 触发：On Click
   → 动画：Push Left (左滑)

3. 测试原型
   → 点击右上角 "▶ Play"
   → 点击按钮看效果

💡 实用技巧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

技巧1：复用组件
  → 设计一个按钮
  → 右键 → "Create component"
  → 拖拽使用多次
  → 修改主组件，全部同步

技巧2：自动布局
  → 选择多个元素
  → 右键 → "Add flex layout"
  → 设置间距和对齐
  → 自动适应内容

技巧3：使用网格
  → View → Show grid
  → 设置8px网格
  → 对齐更精准

技巧4：快速样式
  → 选择元素
  → 右侧面板调整样式
  → 保存为"Style"
  → 一键应用到其他元素
