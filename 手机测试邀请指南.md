
```
http://192.168.1.100:8501
```

## 🔍 详细操作步骤

### Mac端配置（主机）

#### 1. 获取本机IP地址

**方法A：命令行（推荐）**
```bash
# 查看所有网络接口
ifconfig

# 或只看IP地址
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**输出示例：**
```
inet 192.168.1.100 netmask 0xffffff00 broadcast 192.168.1.255
```

你的IP地址就是：`192.168.1.100`

---

**方法B：图形界面**
1. 点击Apple菜单 () → "系统偏好设置"
2. 点击"网络"
3. 选择正在使用的连接（WiFi或以太网）
4. 右侧显示"IP地址"

---

#### 2. 检查Streamlit运行状态

**确认系统正在运行：**
```bash
# 查看进程
ps aux | grep streamlit

# 应该看到类似：
# streamlit run app.py
```

**检查端口：**
```bash
# 查看8501端口
lsof -i :8501 2>/dev/null || netstat -an | grep 8501
```

---

#### 3. 检查防火墙设置

**Mac防火墙配置：**
1. 打开"系统偏好设置" → "安全性与隐私"
2. 点击"防火墙"标签
3. 如果防火墙已开启：
   - 点击"防火墙选项"
   - 确保Python/Streamlit被允许接收连接
   - 或临时关闭防火墙测试

**快速测试（临时关闭防火墙）：**
```bash
# 关闭防火墙（需要管理员权限）
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setglobalstate off

# 测试完成后重新开启
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setglobalstate on
```

## 🐛 常见问题解决

### 问题1：手机无法访问

**症状：** 浏览器显示"无法连接到服务器"

**解决方案：**

**A. 检查网络**
```bash
# 在Mac上测试同事手机的IP是否可达
ping [同事手机IP]
```

**B. 检查防火墙**
- Mac系统偏好设置 → 安全性与隐私 → 防火墙
- 临时关闭防火墙测试

**C. 重启Streamlit**
```bash
# 停止
ps aux | grep streamlit | grep -v grep | awk '{print $2}' | xargs kill

# 重新启动
cd ~/Desktop/销售AI情报系统
streamlit run app.py --server.address=0.0.0.0
```

**重要：** 添加 `--server.address=0.0.0.0` 允许外部访问！

---

### 问题2：页面加载很慢

**原因：** 手机网络速度或Mac性能

**解决方案：**
- 确保Mac和手机都连接5G WiFi（不是2.4G）
- 关闭Mac上其他占用资源的程序
- 清除手机浏览器缓存

---

### 问题3：页面显示不正常

**症状：** 布局错乱或功能按钮点击无反应

**解决方案：**
- 刷新页面（下拉刷新）
- 切换到横屏模式
- 尝试不同浏览器
- 清除浏览器缓存

---

### 问题4：文件上传失败

**症状：** 上传照片或文档后没有反应

**解决方案：**
- 检查文件大小（限制200MB）
- 确认文件格式（PDF、JPG、PNG）
- 尝试压缩文件后上传
- 切换到WiFi（不要用移动数据）

## 🔒 安全建议

### 生产环境部署（可选）

**如果需要长期使用或外网访问：**

#### 方式1：内网穿透（简单）

**使用ngrok：**
```bash
# 安装ngrok
brew install ngrok

# 启动内网穿透
ngrok http 8501
```

**会得到一个公网地址：**
```
https://abc123.ngrok.io
```

**优点：**
- 任何人都能访问（无需同一WiFi）
- 支持HTTPS
- 简单快速

**缺点：**
- 免费版URL会变
- 有一定安全风险

---

#### 方式2：VPS部署（专业）

**部署到云服务器：**
1. 租用云服务器（阿里云、腾讯云）
2. 配置域名和HTTPS
3. 部署系统
4. 设置访问控制

**优点：**
- 稳定可靠
- 支持多人同时访问
- 完整的安全控制

---

### 访问控制

**添加密码保护（可选）：**

创建 `.streamlit/config.toml`：
```toml
[server]
address = "0.0.0.0"
port = 8501

[passwords]
password = "your_password_here"
```
