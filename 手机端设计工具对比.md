## 🎯 你可能在找的工具

### 1. **Pencil Project** (铅笔项目)

**官网：** https://pencil.evolus.vn/

**简介：**
- 免费开源的GUI原型设计工具
- 支持Windows、Mac、Linux
- 内置移动应用原型模板

**特点：**
- ✅ 完全免费
- ✅ 开源软件
- ✅ 支持导出HTML、PDF、PNG
- ✅ 内置移动UI组件库
- ❌ 不支持在线协作
- ❌ 功能相对基础

**适合场景：**
- 快速原型设计
- 个人项目
- 离线工作

---

### 2. **Penpot** (笔盆 - 可能是你说的"最近出的")

**官网：** https://penpot.app/

**简介：**
- 开源的设计与原型工具
- 基于Web，无需安装
- 2021年推出，快速流行

**特点：**
- ✅ 完全免费开源

- ✅ 支持团队协作
- ✅ 类似Figma的现代界面
- ✅ 支持SVG原生格式
- ✅ 自托管选项

**适合场景：**
- 团队协作设计
- 专业UI/UX设计
- 替代Figma的开源方案

## 🚀 Penpot 快速上手指南

### 第一步：访问Penpot

**方式1：在线使用（推荐）**
```
https://design.penpot.app/
```

**方式2：自托管（高级）**
```bash
docker run -d \
  -p 9001:80 \
  penpotapp/frontend:latest
```

---

### 第二步：注册账号

1. 访问 https://design.penpot.app/
2. 点击"Sign up"注册
3. 输入邮箱和密码
4. 验证邮箱

**或者：** 使用Google账号快速登录

---

### 第三步：创建第一个项目

#### 1. 创建新项目

点击"Create new project"

**项目设置：**
- 名称：销售AI情报系统原型
- 类型：Mobile App

#### 2. 创建画板（Artboard）

**选择手机尺寸：**
- iPhone 14 Pro: 393×852
- iPhone 14 Pro Max: 430×932
- Android (标准): 360×800

**快捷键：**
- 按 `A` 键创建画板
- 选择预设尺寸

---

### 第四步：设计界面

#### 基础操作

**工具栏（左侧）：**
- `V` - 选择工具
- `R` - 矩形
- `O` - 圆形
- `T` - 文本
- `F` - Frame（容器）

**组件库（左侧面板）：**
- 点击"Assets"查看组件
- 拖拽组件到画板

---

#### 设计移动界面步骤

**1. 设计顶部导航栏**
```
1. 创建矩形（R）
2. 宽度：393px，高度：88px
3. 填充颜色：#1E40AF (蓝色)
4. 添加文本（T）："销售AI情报系统"
5. 文本颜色：白色
```

**2. 设计标签栏**
```
1. 创建9个标签按钮
2. 使用自动布局（Auto Layout）
3. 间距：8px
4. 图标 + 文字组合
```

**3. 设计内容区域**
```
1. 根据功能设计表单
2. 添加输入框、按钮
3. 使用组件重用
```

---

### 第五步：创建交互原型

#### 添加链接

1. **进入原型模式**
   - 点击右上角"Prototype"标签

2. **创建链接**
   - 选择一个按钮
   - 拖拽蓝色圆点到目标页面
   - 设置交互触发：On Click
   - 设置动画：Dissolve（淡入淡出）

3. **设置动画**
   - Push Left（左滑）
   - Push Right（右滑）
   - Dissolve（淡入淡出）
   - Instant（立即切换）

---

### 第六步：手机预览

#### 方式1：浏览器预览

1. 点击右上角"Play"按钮
2. 会打开预览窗口
3. 点击交互测试流程

**获取分享链接：**
```
1. 点击"Share prototype"
2. 复制链接
3. 在手机浏览器打开
4. 添加到主屏幕（像真实APP）
```

---

#### 方式2：二维码分享

1. 点击"Share"
2. 生成二维码
3. 手机扫码访问
4. 测试交互流程
